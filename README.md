# Hospital Management System

A backend Hospital Management System developed using **ASP.NET Core Web API** and **SQL Server**.
This project manages hospital operations such as patient records, doctor management, appointments, and departments.

## 🚀 Features

* Patient Management
* Doctor Management
* Department Management
* Appointment Scheduling
* Authentication & Authorization
* RESTful APIs
* CSV Export for Appointments
* Error Handling & Logging
* Production-ready architecture

## 🛠️ Technology Stack

* ASP.NET Core Web API (.NET)
* C#
* SQL Server
* Entity Framework Core
* SignalR
* Swagger API Documentation

## 📂 Project Structure

```
HospitalMS.API
HospitalMS.DATA
HospitalMS.Services
HospitalMS.Models
```

## 📊 Modules

### Patient Module

* Register Patient
* Update Patient Information
* View Patient Details

### Doctor Module

* Add Doctor
* Assign Department
* Manage Doctor Information

### Appointment Module

* Schedule Appointment
* Update Appointment Status
* Export Appointment Data

### Department Module

* Add Departments
* Manage Department Details
